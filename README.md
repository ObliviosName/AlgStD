AlgStD

Этот репозиторий содержит объединённые лабораторные работы по курсу "Алгоритмы и структуры данных". Проект позволяет запускать все лабораторные из одного исполняемого файла с помощью меню выбора.

Лабораторные работы

Лабораторная 1: Подсветка ключевого слова double в C-коде.

Лабораторная 2: Сортировка массивов (методы Шелла и слияния) с вводом через клавиатуру, генерацией или файлом.

Лабораторная 3: Работа со связными списками (создание, поиск особого элемента).

Лабораторная 4: Дерево AVL (построение из файла, удаление слов по длине, отображение дерева).

Лабораторная 5: Алгоритмы на графах (Дейкстра и Флойд–Уоршелл для поиска кратчайшего пути).

Как запустить

Склонируйте репозиторий:

git clone https://github.com/ObliviosName/AlgStD.git
cd AlgStD


Скомпилируйте с помощью Makefile:

make


Или вручную:

gcc -Wall -Wextra -O2 -o main main.c src/lab1/lab1.c src/lab2/lab2.c src/lab3/lab3.c src/lab4/lab4.c src/lab4/printtree.c src/lab5/lab5.c


Запустите:

./main


Следуйте меню для выбора лабораторной работы.

Для лабораторных, где требуется файл данных, указывайте путь относительно папки src/lab#.

Структура проекта
algst/
│-- main.c          # Основной файл с меню
│-- makefile        # Makefile для компиляции
│-- README.md
└-- src/
    ├-- lab1/      # Код и данные для лабораторной 1
    ├-- lab2/      # Код и данные для лабораторной 2
    ├-- lab3/      # Код для лабораторной 3
    ├-- lab4/      # Код и данные для лабораторной 4 (включая printtree.c)
    └-- lab5/      # Код для лабораторной 5

Требования

Компилятор GCC (MinGW или MSYS2 для Windows).

Make (если используете Makefile).

Файлы данных (input.txt и др.) находятся в соответствующих папках src/lab#.
